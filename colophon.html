Colophon

Over the last few years, my time spent online has decreased and become more conscious. Maybe it was inevitable to stumble across a smaller web. It was from this discovery that I began to write code for the first time. I found that, in a new way, I was finally getting my teeth back into design.


This website was built and is maintained by hand using HTML, CSS and a little JavaScript. I use VS Studio Code as an editor. The site is hosted on GitHub Pages. All very grassroots.


The typeface is Noto Sans Mono.


I am still very new to this whole web design thing. If you come across something that seems broken, feel free to let me know!


Thanks for visiting this space. See you around.


- Eddie
